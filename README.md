# Nexus Digital — Marketing Agency Website

A full-stack digital marketing agency website — single-page frontend (HTML/CSS/JS) backed by a Node.js + Express server that saves form submissions (custom plan requests, complaints) into Excel files.

## Features
- Single-page app with smooth section navigation, no reloads
- Five-tier pricing packages with interactive cards
- Custom plan request form and customer complaint form, both backed by a real API
- Fully responsive design
